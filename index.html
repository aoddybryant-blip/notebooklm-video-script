CLIP 2 What is NotebookLM?
0:08 – 0:16 ▼
Visual Description
ทรงกลม AI เรืองแสงสีฟ้าลอยอยู่กลางจอ ไอคอนเอกสาร 3 ชนิด (PDF สีแดง, Document สีน้ำเงิน, Slides สีเหลือง) ลอยเข้าหาทรงกลมพร้อม motion trail เมื่อเอกสารถูกดูดเข้าไป จะสลายเป็นอนุภาคข้อมูลสีฟ้า ฝั่งขวาจะมีการ์ดข้อมูลที่จัดระเบียบแล้วค่อยๆ ปรากฏออกมา
Narration Script
Thai
"NotebookLM คือ AI จาก Google ที่อ่านเอกสารของคุณ แล้วตอบจากข้อมูลจริง ไม่มั่ว ไม่แต่ง"
English
"NotebookLM is an AI from Google that reads your documents and answers from real data. No hallucinations. No fabrication."

TimeAction

CLIP 3 Sales Prompts Highlight
0:16 – 0:24 ▼
Visual Description
ในห้อง Command Center สไตล์ Cyberpunk การ์ด Holographic 3 ใบปรากฏทีละใบ: การ์ดม่วง (โล่ = Objection Rebuttal), การ์ดฟ้า (ดาบไขว้ = Battlecard), การ์ดเขียว (เป้า = Closing Technique) อนุภาคแสงไหลเชื่อมระหว่าง 3 การ์ด
Narration Script
Thai
"มี Prompt สำหรับ Sales ถึง 11 ตัว ตั้งแต่โต้ข้อโต้แย้ง เตรียม Battlecard ไปจนถึงปิดการขาย"
English
"There are 11 Sales Prompts — from handling objections, preparing Battlecards, all the way to closing deals."
Timing Breakdown
TimeAction
0:16การ์ดม่วง (Objection Rebuttal) ปรากฏฝั่งซ้าย
0:17เริ่มพูด "มี Prompt สำหรับ Sales ถึง 11 ตัว..."
0:19การ์ดฟ้า (Battlecard) ตกลงมาจากด้านบน ลงตรงกลาง
0:20"...ตั้งแต่โต้ข้อโต้แย้ง เตรียม Battlecard..."
0:22การ์ดเขียว (Closing) ซูมเข้ามาจากด้านหลัง ลงฝั่งขวา
0:23"...ไปจนถึงปิดการขาย" — อนุภาคแสงเชื่อม 3 การ์ด
Speaker Notes
พูดตัวเลข "11 ตัว" ให้ชัดเจน เพื่อสร้างความประทับใจว่ามีเครื่องมือครบ ไม่ต้องอธิบายทุก Prompt แค่ highlight 3 ตัวที่สำคัญที่สุด: โต้แย้ง, เตรียมสู้คู่แข่ง, ปิดการขาย — ครอบคลุมทั้ง Sales Cycle
CLIP 4 Marketing & Strategy
0:24 – 0:32 ▼
Visual Description

"Covering both Marketing and Strategy — from crafting Value Propositions to Executive Summaries."

CLIP 5 Time Savings
0:32 – 0:40 ▼
Visual Description
หน้าจอแบ่งครึ่ง: ฝั่งซ้ายสีเทา (ออฟฟิศแบบเดิม) มีนาฬิกาแขวนผนังและกองเอกสาร vs ฝั่งขวาสีนีออน (AI Interface) มีวงกลม Progress 0%→100% เส้นแบ่งสีฟ้าเรืองแสงค่อยๆ เลื่อนไปทางซ้าย ฝั่ง AI ขยายใหญ่ขึ้น เมื่อถึง 100% เครื่องหมายถูกสีเขียวปรากฏ
Narration Script
Thai
"ประหยัดเวลาได้กว่า 7 ชั่วโมงต่อดีล เปลี่ยนเวลาที่เสียไปกับเอกสาร มาปิดการขายแทน"
English
"Save over 7 hours per deal. Turn time wasted on paperwork into time spent closing sales."
Timing Breakdown
TimeAction
0:32Split screen ปรากฏ — ฝั่งซ้ายเทา, ฝั่งขวานีออน
0:33เริ่มพูด "ประหยัดเวลาได้กว่า 7 ชั่วโมงต่อดีล..."
0:35Progress meter เริ่มเพิ่มขึ้น, กองเอกสารฝั่งซ้ายสูงขึ้น
0:37"...เปลี่ยนเวลาที่เสียไปกับเอกสาร..." — เส้นแบ่งเลื่อนซ้าย
0:39"...มาปิดการขายแทน" — 100% + เครื่องหมายถูกเขียว
Speaker Notes
ตัวเลข "7 ชั่วโมง" เป็น key message ที่ต้องพูดให้ชัดและหนักแน่น ภาพ split screen ทำให้เห็น contrast ชัดเจนระหว่างวิธีเก่ากับวิธีใหม่ จบด้วยการ reframe — ไม่ใช่แค่ "ประหยัดเวลา" แต่ "เอาเวลาไปปิดการขาย" ซึ่งเป็นสิ่งที่ Sales ทุกคนอยากได้
CLIP 6 Call to Action (CTA)
0:40 – 0:48 ▼
Visual Description
พื้นหลังมืดสไตล์ Cyberpunk พื้นที่ชื่อเรื่องสีขาวปรากฏด้านบน แถบนีออนสีเขียวค่อยๆ ก่อตัวจากซ้ายไปขวาพร้อมอนุภาคแสง เมื่อแถบสมบูรณ์จะมีแสง starburst ไอคอน 3 วงกลมปรากฏทีละอัน: ลูกโลกสีม่วง (เข้าเว็บ), คลิปบอร์ดสีฟ้า (Copy), จรวดสีเขียว (เริ่มใช้งาน)

Thai
"เข้าเว็บไซต์ Copy Prompt แล้วเริ่มใช้งานได้ทันที วันนี้เลย"
English
"Visit the website, copy the prompts, and start using them right away. Today."
Timing Breakdown
TimeAction
0:40พื้นที่ชื่อเรื่องปรากฏด้านบน
0:41เริ่มพูด "เข้าเว็บไซต์..." — แถบเขียวเริ่มก่อตัว

0:44"...แล้วเริ่มใช้งานได้ทันที..." — ไอคอนจรวดเขียวปรากฏ
0:46"วันนี้เลย" — แสง starburst, อนุภาคลอย
0:48BGM fade out, จบวิดีโอ
Speaker Notes
CTA ต้องชัดเจนและง่าย: 3 ขั้นตอน (เข้าเว็บ → Copy → ใช้งาน) จบด้วย "วันนี้เลย" เพื่อสร้าง urgency ทิ้งจังหวะหลัง "วันนี้เลย" ให้ BGM fade out อย่างสวยงาม
